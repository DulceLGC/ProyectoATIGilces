PREGUNTAS

1. ¿Qué es GitHub?

GitHub es el mayor proveedor de alojamiento de repositorios Git, y es el punto de encuentro para que millones de desarrollados colaboren en el desarrollo de sus proyectos. Un gran porcentaje de los repositorios Git se almacenan en GitHub, y muchos proyectos de codigo abierto lo utilizan para hospedar su Git, realizar su seguimiento de fallas, hacer revisiones de codigo y otras cosas.

2. ¿Cómo crear la cuenta en GitHub paso a paso?

  • Ingresar a https://github.com
  • Elegir un nombre de usuario que no esté ya en uso
  • Proporciona un correo y una contraseña
  • Pulsar el botón verde grande "Sign up for GitHub"
  • GitHub te enviará un correo para verificar la dirección que le has dado, confirmar la direccion
  • Para ir al escritorio principal pulsar el logo del gato con patas de pulpo


1. ¿Qué es un Hosting?

Un hosting, o alojamiento web, es un servicio que permite almacenar y distribuir los archivos de una página web en Internet, haciéndola accesible a cualquier usuario. Cuando contratas un servicio de hosting, básicamente alquilas un espacio en un servidor físico donde puedes almacenar todos los archivos y datos necesarios para que tu sitio web funcione correctamente.

2. Ejemplos de Hostings

  • Wix: Wix.com es una plataforma de creación de sitios web que también ofrece servicios de hosting.

  • Bluehost: Bluehost es un proveedor de hosting compartido, VPS y servidores dedicados.

  • HostGator: HostGator es otro proveedor popular de hosting compartido, VPS y servidores dedicados.

  • GoDaddy: GoDaddy es un registrador de dominios y también ofrece servicios de hosting.

  • Hostinger: Hostinger es un proveedor de hosting asequible, conocido por sus planes de hosting compartido.

  • A2 Hosting: Ofrece una variedad de opciones de hosting, incluyendo compartido, VPS, y dedicado.

  • SiteGround: Es conocido por su soporte técnico de alta calidad y sus servicios de hosting optimizados para WordPress.

  • DreamHost: Ofrece una variedad de planes de hosting, incluyendo compartido, VPS, y dedicado.

  • Google Cloud: Es un servicio de hosting en la nube, que ofrece una gran flexibilidad y escalabilidad.

  • Squarespace: No solo es un servicio de hosting, sino también un creador de sitios web con una interfaz fácil de usar.
